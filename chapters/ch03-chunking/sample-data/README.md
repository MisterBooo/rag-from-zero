# 测试样本

本目录用于放你自己的测试 PDF。

## 怎么准备

`chunk_demo.py` 默认读取当前目录的 `your_doc.pdf`。你可以:

- **用你手头任意一份 PDF**:保险条款、产品说明书、论文、报告都行,几页到几十页皆可。把它命名为 `your_doc.pdf` 放到 `chapters/ch03-chunking/` 目录即可。
- **想要保险场景的样本**:可以从公开渠道下载任意一份保险产品条款 PDF(各保险公司官网通常都能下到产品条款),体验"条款被切碎"的真实感会更强。

> 出于版权与隐私考虑,本仓库不附带真实保险客户文档。`reproduce-disaster.py` 不需要任何 PDF,直接 `python reproduce-disaster.py` 就能跑。
